# 啟示錄紅龍 | TheGreatRedDragon
這是由 **CDD 團隊** 在Google Gemini AI協助下自主開發的 Discord TRPG 跑團專用擲骰機器人。

## 核心擲骰功能
- **一般單次擲骰**：支援如 `1d100` 或 `2d6+3` 的算式。 
- **複雜算式解析**：支援括號與乘法計算，例如 `2(1d6+3)`，並會詳細顯示每顆骰子的點數。 
- **複數擲骰 (具備互動按鈕)**：
    - 使用 `!r 次數 算式` (如 `!r 3 1d10`) 時，機器人會產生帶有顏色區分的按鈕。 
    - 使用者點擊按鈕後，才會顯示詳細的擲骰過程，避免洗版。 
- **直接顯示版複數擲骰**：使用格式如 `3 1d100` 可直接列出多次結果。 

## CoC 7 版專用檢定
- **CC 技能檢定**： 
    - 自動判定成功等級：★大成功、極限成功、困難成功、普通成功、失敗、✘大失敗。 
    - 支援獎勵骰與懲罰骰 (`!ccn`)。 
    - 支援聯合檢定 (可一次輸入多個目標值)。 
- **SC 理智檢定**：專用格式 `!sc 成功率 成功損害/失敗損害`，會自動計算檢定結果並根據成功與否扣除理智點數。 
- **CG 成長檢定**：支援單個或多個技能的成長判定，成功時會自動擲 `1D10` 並計算新數值。 
- **狂氣症狀查詢**：
    - `!ccrt`：隨機抽取「即時症狀」並擲出持續輪數。 
    - `!ccsu`：隨機抽取「總結症狀」並擲出持續小時。 

## 系統與管理功能
- **隨機選擇**：輸入 `隨機 選項1 選項2` 讓機器人幫你做決定。 
- **雙介面支援**：所有主要功能同時支援 **傳統訊息指令** (以 `!` 開頭) 與 **斜線指令** (`Slash Commands`)。 
- **Windows 系統匣整合**：具備圖標顯示功能，可透過電腦右下角的系統匣直接關閉機器人。
